# 📝 NoteKeeper Clone

A simple and elegant note-taking web application built with vanilla HTML, CSS, and JavaScript. Store your notes locally in your browser with a beautiful gradient background.

![NoteKeeper Demo](demo.gif) <!-- Replace with your actual demo GIF -->

## ✨ Features

- ➕ **Add Notes**: Create new notes with title and description
- ✏️ **Edit Notes**: Update existing notes easily
- 🗑️ **Delete Notes**: Remove notes with confirmation
- 💾 **Local Storage**: Notes are saved in your browser's local storage
- 🎨 **Beautiful UI**: Modern design with gradient background and smooth animations
- 📱 **Responsive**: Works on different screen sizes

## 📋 How to Use

1. **Clone the repository**:

   ```bash
   git clone https://github.com/your-username/notekeeper-clone.git
   ```

2. **Navigate to the project directory**:

   ```bash
   cd notekeeper-clone
   ```

3. **Open the application**:
   - Open `index.html` in your web browser
   - Or use a local server (recommended for better experience):

     ```bash
     # Using Python
     python -m http.server 8000

     # Using Node.js (if you have http-server installed)
     npx http-server
     ```

4. **Start taking notes!**
   - Click the "Add new note" box to create a note
   - Fill in the title and description
   - Click "Add" to save
   - Use the menu (three dots) to edit or delete notes

## 🎯 Key Features Explained

### Local Storage Integration

- Notes are automatically saved to your browser's local storage
- No data is sent to any server - everything stays on your device
- Notes persist even after closing the browser

### User-Friendly Interface

- Intuitive add/edit/delete functionality
- Confirmation dialog for deletions
- Smooth popup animations
- Responsive grid layout
